# LockCheck

LockCheck is a lightweight World of Warcraft addon that helps reduce summon
coordination friction by showing Warlock presence and incoming summons.

## Features
- Detects Warlock presence in party/raid
- Shows incoming summons and summoner name
- Manual summon request / thank-you button
- Configurable summon request method: whisper Warlock or send to raid/party chat
- Automatically hidden in combat

## Commands
- `/lockcheck reset` - Reset the frame position
- `/lockcheck mode whisper` - Set summon requests to whisper the Warlock (default)
- `/lockcheck mode group` - Set summon requests to send to raid/party chat
- `/lockcheck` - Show current mode and available commands
