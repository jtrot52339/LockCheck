# LockCheck

LockCheck is a lightweight World of Warcraft addon that helps reduce summon
coordination friction by showing Warlock presence and incoming summons.

## Features
- Detects Warlock presence in party/raid
- Shows incoming summons and summoner name
- Manual summon request / thank-you button
- Automatically hidden in combat
